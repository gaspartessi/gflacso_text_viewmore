ABOUT THIS ACTIVITY MODULE
=========================================
Permite agregar etiquetas pero con una descripción corta y descripción larga


INSTALL INSTRUCTIONS
=========================================
Create a directory called "gflacso_text_viewmore" in your "blocks" directory and copy all the files for this module into the "gflacso_text_viewmore"
directory.  Log in to your Moodle site as an administrator and click on the "notifications" link in the Adminsitration block and
ensure all tables were setup correctly.
